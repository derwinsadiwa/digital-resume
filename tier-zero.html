Built a tier zero interactive landing pages

  • Live Demo

Tier Zero is a collection of blog articles where we transform static pages to interactive pages like adding parallax that includes fixed images in the background that is kept in place and user can scroll down the page to see different parts of the images.

Also added scrollbased driven approach whereas the animation will trigger once the user scroll down the page. It has a lot of features of scroll trigger like pin elements in place, scrub through animations, supports vertical and horizontal scrolling and gsap integration which is a popular and standard for javascript animation nowadays.

  • Design the mockup using Figma and Adobe XD
  • Responsive layout on desktop, mobile and other devices
  • SEO Support for traffic and conversion
  • Scroll driven animations using ScrollTrigger
